# 簡易電卓 (Simple Calculator)

## 概要
HTML、CSS、JavaScriptを使用して作成したシンプルな電卓アプリケーションです。

## 機能
- 四則演算（加算、減算、乗算、除算）
- 小数点計算対応
- クリア機能
- バックスペース機能（一文字削除）
- キーボード入力対応
- レスポンシブデザイン

## ファイル構成
```
simple-calculator-js/
├── index.html      # メインHTMLファイル
├── style.css       # スタイルシート
├── script.js       # JavaScript機能
└── README.md       # このファイル
```

## 使用方法
1. `index.html`をブラウザで開く
2. ボタンクリックまたはキーボード入力で数値と演算子を入力
3. `=`ボタンまたはEnterキーで計算実行

## キーボードショートカット
- `0-9`, `.`: 数値・小数点入力
- `+`, `-`, `*`, `/`: 演算子入力
- `Enter`, `=`: 計算実行
- `Escape`, `C`: クリア
- `Backspace`: 一文字削除

## 技術スタック
- HTML5
- CSS3 (Grid Layout, Flexbox, レスポンシブデザイン)
- Vanilla JavaScript (DOM操作)
